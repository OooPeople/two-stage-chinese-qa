# Sample Data Notes

This directory contains small examples extracted from the original coursework data so the repository can demonstrate the expected file formats without redistributing the full dataset.

Files included here:

- `context_sample.json`
- `train_sample.json`
- `valid_sample.json`
- `test_sample.json`
- `sample_submission.csv`

Important notes:

- Each split contains only a small number of examples.
- The paragraph indices in the sample split files were remapped to match `context_sample.json`.
- These files are for format illustration and quick smoke tests only.
- If the original dataset license does not allow redistribution, remove this directory before publishing.
